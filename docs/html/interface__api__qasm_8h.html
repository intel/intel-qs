unsigned long ExecuteHandler (std::string op, std::string args)
 

Detailed Description

This file contains the interface to the QASM command handler function. The QASM operation is broken up into a command token and an args token. The command is mapped to a hash table of functions designed to handle the QASM op and then the function is invoked and a result returned.
